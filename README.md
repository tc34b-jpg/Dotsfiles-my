# 📂 Meus Dots | Setup Hyprland 🐧

Este é o meu setup pessoal. Se você quer deixar o seu sistema com a minha cara, siga os passos abaixo!

## 📦 O que você vai precisar (Dependências)
- **Interface:** Hyprland, Waybar, Rofi, Wlogout
- **Shell:** Fish, Starship (prompt)
- **Fontes:** JetBrainsMono Nerd Font

## 🚀 Como instalar (Para o meu Pai ❤️)
1. Abra o terminal.
2. digite:
```
    git clone https://github.com/tc34b-jpg/Dotsfiles-my.git
   
   cd Dotsfiles-my/dotfiles
   
   chmod +x install.sh

   ./install.sh
```
